Google Workspace Sync (BYOC)

1. Enable APIs in Google Cloud Console

Go to your Google Cloud Console, create a new project, and enable the Google Calendar API, Google Drive API, Google Tasks API, and Google Docs API inside the API Library.

2. Configure OAuth Consent Screen

Set User Type to "External", publish the app state to Testing, and add your own Google email address as a test user to authorize secure, personal OAuth connections.

3. Create Desktop OAuth Credentials

Navigate to Credentials, click "Create Credentials" -> "OAuth client ID", and set Application Type to **Desktop app**. Secure the client credentials keys:

ClientID: xxxxx-xxxxx.apps.googleusercontent.com
ClientSecret: GOCSPX-xxxxxxxxxxxxx

4. Link Credentials inside Reality Settings

Enter the generated ClientID and ClientSecret inside Reality's Settings panel.Reality will sync and format daily metrics sheets inside your Google Drive, and map tasks directly into Google Tasks.
