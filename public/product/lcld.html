# 🛡️ LCLD: Locate & Control Lost Device [![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0) [![Android](https://img.shields.io/badge/Platform-Android-green.svg)](https://developer.android.com) [![Downloads](https://img.shields.io/github/downloads/pawanwashudev-official/LCLD/total.svg)](https://github.com/pawanwashudev-official/LCLD/releases) [![Open Source](https://img.shields.io/badge/Open%20Source-%E2%9D%A4-red.svg)](https://github.com/pawanwashudev-official/LCLD) **LCLD (Locate & Control Lost Device)** is the ultimate privacy-focused, offline-first security solution for your Android device. We've taken the core foundation of open-source tracking and completely overhauled it for the modern user. Developed by **Pawan Washudev** ([@pawanwashudev-official](https://github.com/pawanwashudev-official)) --- ## 🚀 Why LCLD is Better Than Default FMD Apps Standard FMD apps are often tied to a single ecosystem, require an active internet connection, and are easily bypassed. **LCLD is built to be unstoppable.** | Feature | Default FMD Apps | **LCLD** | | :--- | :---: | :---: | | **Offline Control** | ❌ (Requires Data/WiFi) | ✅ **Full Control via SMS** | | **Unstoppable Siren** | ⚠️ Easy to Mute | ✅ **Forces 100% Vol + Lock Required** | | **Theft Mode** | ❌ Basic Tracking | ✅ **Persistent Multi-Action Protocol** | | **Server Connection** | ⚠️ Needs Extra Apps | ✅ **Native Push (No UnifiedPush needed)** | | **UI/UX** | ⚠️ Functional but Dated | ✅ **Elegant & Simplified Setup** | | **Privacy** | ⚠️ Tracked by Big Tech | ✅ **100% Private / Self-Hostable** | --- ## ✨ Key Features & Major Enhancements ### 🆕 Elegant & Simplified UI We've completely redesigned the interface to be beautiful, intuitive, and modern. Setup is now a breeze—go from installation to full protection in under 60 seconds. ### 🚨 Unstoppable Siren (Enhanced Ring) The `ring` command is now a true siren. - **Auto-Max Volume:** Instantly forces device volume to 100%. - **Anti-Mute:** If a thief tries to lower the volume, LCLD automatically pulls it back to maximum. - **Unlock to Stop:** The siren **will not stop** until the device is successfully unlocked by the owner. ### ⚡ Integrated Server Support Unlike the original FMD, **LCLD does not require an additional app** (like Push Distributor) to receive server commands. We've built the communication layer directly into the app for a lighter, more reliable experience. ### 🛑 THEFT MODE (The Ultimate Countermeasure) The new `theft` command is a proactive security protocol that triggers multiple actions simultaneously: - **Instant Lockdown:** Locks the device immediately. - **Continuous Tracking:** Starts aggressive location reporting. - **Siren Activation:** Triggers the unstoppable siren. - **Persistence:** Even if the thief restarts the phone, the siren and lock will resume **instantly** upon boot. There is no escape. ### 📸 Remote Camera & Precision Locating - **Remote Snap:** Capture high-quality photos from front/back cameras. - **GPS + Cell Tower:** Locate your device even when GPS is disabled or signal is weak. --- ## 🛠️ Getting Started 1. **Download:** Get the latest APK from [Releases](https://github.com/pawanwashudev-official/LCLD/releases). 2. **Setup PIN:** Set your secure 4-6 digit PIN for SMS commands. 3. **Grant Permissions:** Enable Location, Camera, and Device Admin for full protection. 4. **Trusted Contacts:** (Optional) Add numbers that can control the device without a PIN. --- ## 🔒 Security & Privacy LCLD is built with a **Security-First** mindset. - **No Tracking:** We don't track your location. Your data is only shared with the channels *you* configure. - **Encrypted Communication:** Server communication is secured via industry-standard encryption. --- ## ❤️ Credits & Foundation LCLD is a heavily enhanced and rebranded version of the **Find My Device (FMD)** app by **Nulide**. We've built upon their excellent FOSS foundation to create the most powerful device control app available. - **Original Project:** [FMD Android](https://gitlab.com/fmd-foss/fmd-android) - **Original Developer:** [Nulide](https://fmd-foss.org) --- **Developed with ❤️ by Pawan Washudev**
